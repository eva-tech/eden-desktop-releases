# Eden Desktop — Releases

Public distribution + auto-update artifacts for **Eden Desktop** (`.dmg`, `.zip`, `latest-mac.yml`, `.blockmap`).

- These are **signed + notarized** installer binaries, published automatically by CI on each release.
- **This repository contains NO source code.** The source is private in `eva-tech/eden-desktop`.
- Downloading an installer grants no access — the application requires login to the Eden backend.
